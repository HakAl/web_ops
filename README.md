# Web Ops

The Web Ops team manages the blog and public site for the [Skills Team](https://github.com/HakAl/team_skills) project.

**Live site:** [vibecoder.buzz](https://vibecoder.buzz/)

## What's here

```
docs/           # Published site (GitHub Pages source)
  blog/         # Blog posts (HTML + markdown source)
.beads/         # Issue tracking (br)
.team/          # Cross-team dispatch and design docs
```

## The team

| Role | Persona | What they do |
|------|---------|--------------|
| Creative Director | Dana | Drives site strategy, content planning, publishing cadence |
| Design Critic | Dee | Challenges design choices, ensures UX consistency |
| Frontend Developer | Blake | Turns approved designs into working, accessible pages |
| Content Writer | Nora | Writes compelling copy, maintains voice consistency |
| SEO/Analytics | Ari | Measures performance, SEO, surfaces insights |
| QA/Accessibility | Grace | Validates everything before it ships |

## Workflow

```
User Request → Dana (plan) → Dee (critique) → Blake (build) → Grace (validate) → Ship
```

## Getting started

This repo is designed for AI coding agents with the [team_skills](https://github.com/HakAl/team_skills) personas installed. Skills are a standardized format — works with Claude Code, Codex, and any agent that supports them.

```bash
# Clone
git clone <repo-url>
cd _web_ops

# Issue tracking
br onboard
br ready
```

See [AGENTS.md](AGENTS.md) for agent instructions and [TEAM.md](TEAM.md) for team protocols.
